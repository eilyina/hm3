1.В IntelliJ IDEA открыть проект.
2.Запустить проект Run 'ExelWriter'.

Программа создает файл формата .xls в директории C:\Тест\Тест.xls с данными о пользователях,полученных
из API https://randomuser.me/, если получен и обработан ответ, или
из текстовых файлов расположенных в /resources текущего проекта.
